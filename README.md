# 🏦 Caixa Eletrônico COBOL

## 💾 Simulação de Sistema Bancário utilizando GnuCOBOL

Este projeto apresenta uma simulação de um **Caixa Eletrônico desenvolvido em COBOL**, utilizando o compilador **GnuCOBOL (cobc)** e execução via terminal.

A aplicação foi criada com o objetivo de praticar os fundamentos da linguagem COBOL e compreender a estrutura de desenvolvimento utilizada em sistemas corporativos, especialmente aqueles presentes no setor financeiro e em ambientes **Mainframe**.

O sistema implementa operações bancárias básicas, como **depósito, saque e consulta de saldo**, utilizando estruturas tradicionais da linguagem COBOL, incluindo divisões de programa, armazenamento de dados, controle de fluxo e processamento de informações.

Apesar de ser uma tecnologia criada há décadas, o COBOL continua sendo uma linguagem fundamental em grandes instituições financeiras devido à sua **confiabilidade, estabilidade e capacidade de processar grandes volumes de transações**.

Este projeto representa uma etapa da minha evolução técnica, unindo conhecimentos de desenvolvimento moderno com tecnologias clássicas utilizadas em sistemas críticos.

---

## 🦖 Sobre o COBOL REX

O **COBOL REX** é o mascote deste projeto.

O T-Rex representa a história e a força do COBOL: uma tecnologia considerada "antiga", mas que continua ativa e relevante em bancos, seguradoras e grandes organizações ao redor do mundo.

> 🦖 "Tecnologia clássica, soluções que continuam movimentando o mundo."
