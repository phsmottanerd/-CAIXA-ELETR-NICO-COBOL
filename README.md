# 🏦 Sobre o Projeto

Este projeto consiste em uma **simulação de um Caixa Eletrônico desenvolvido em COBOL**, utilizando **GnuCOBOL**, reproduzindo operações básicas encontradas em sistemas bancários.

O objetivo principal foi desenvolver uma aplicação de terminal aplicando fundamentos da linguagem COBOL, uma tecnologia historicamente presente em bancos, seguradoras, governos e grandes instituições financeiras.

O projeto representa uma etapa de aprendizado voltada para o universo **Mainframe**, explorando a lógica utilizada em sistemas críticos que precisam de estabilidade, segurança e precisão.

---

```

  |  BANKING  |

  +-----------+
```

O COBOL nasceu em 1959 e continua ativo em grandes ambientes corporativos.

Milhões de transações financeiras ainda dependem de sistemas COBOL diariamente.

Características:

✅ Alta confiabilidade
✅ Processamento de grandes volumes de dados
✅ Presença forte no mercado financeiro
✅ Sistemas críticos de missão importante
✅ Integração com ambientes Mainframe

# ▶️ Como Executar

## 1️⃣ Instalar o GnuCOBOL

Tenha o compilador COBOL instalado:

```
GnuCOBOL (cobc)
```

---

## 2️⃣ Compilar o programa

```bash
cobc -x CAIXA_ELETRONICO.cob
```

---

## 3️⃣ Executar

Windows:

```bash
CAIXA_ELETRONICO.exe
```

Linux:

```bash
./CAIXA_ELETRONICO
```
